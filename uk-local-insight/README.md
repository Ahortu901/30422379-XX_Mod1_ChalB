# UK Local Insight (JCS Mod 1b)

Rich web app using UK public APIs:
- DEFRA Flood Monitoring API
- Environment Agency Bathing Water (via /ea proxy/redirect)
- UK Police Data API
- ONS API (beta)

## Tech
- React + Vite
- MUI
- TanStack React Query

## Prerequisites
- Node.js 18+ recommended
